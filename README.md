## Course Syllabus

### Beginner Python
- Variables, String Manipulation, Input/Print
- Data Types, Type Conversion, Math Operations
- Conditionals (if/elif/else), Logical Operators
- Random Module, Error Handling
- Functions, Loops (for/while), Flowcharts
- Lists, Dictionaries, Nested Collections
- Scope, Debugging, Docstrings

### Intermediate Python
- Local Dev Setup (PyCharm), OOP (Classes, Methods, Inheritance)
- Turtle Graphics, Game Development
- File I/O, CSV, Pandas Intro
- List & Dictionary Comprehensions

### Intermediate+
- Tkinter GUI Apps, JSON, SMTP Email
- APIs (Requests, Authentication), Web Scraping (Beautiful Soup)
- Selenium Browser Automation (Tinder, Twitter, LinkedIn)
- Flask Web Development, Jinja2, WTForms
- PythonAnywhere Deployment

- REST APIs, SQLite Databases
- Pandas: Data Cleaning, Pivot Tables, Time Series
- Matplotlib, Seaborn, Plotly Visualizations
- NumPy, Scikit-Learn: Regression, T-Tests
- Jupyter Notebook, Statistical Analysis
